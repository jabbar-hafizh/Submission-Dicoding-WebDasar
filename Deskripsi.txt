Program Sederhana Pendaftaran Ekstrakurikuler Sekolah

Deskripsi :
terdapat file .html .css dan .js
Program memiliki tampilan responsive
Program terdapat pengaplikasian DOM
Program didesain untuk user mendaftar salah satu ekskul

pendaftaran akun selanjutnya akan menghapus akun sebelumnya.
